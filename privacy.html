My Mate Privacy Policy

Last updated: 2026

My Mate is a desktop AI assistant application designed with privacy in mind. This policy explains what data is collected and how it is used.

Information We Collect

  • Email address for authentication
  • Account session data
  • Chat messages and saved memory (if sync is enabled)
  • Files uploaded by the user
  • Subscription status information
  • Basic diagnostic logs

How We Use Information

  • Authenticate users
  • Sync user memory across devices
  • Provide AI responses
  • Process subscriptions
  • Maintain application functionality

Third-Party Services

My Mate uses Supabase (authentication and database), Stripe (payments), and AI processing APIs. These services may process data necessary for application functionality.
